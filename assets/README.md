# 资源文件说明

本目录存放桌面灵宠应用的所有资源文件。

## 目录结构

```
assets/
├── images/          # 图片资源
│   ├── default/     # 默认宠物
│   │   ├── idle.gif      # 待机动画
│   │   ├── walk.gif      # 行走动画
│   │   ├── run.gif       # 奔跑动画
│   │   ├── sleep.gif     # 睡觉动画
│   │   ├── happy.gif     # 开心动画
│   │   ├── sad.gif       # 伤心动画
│   │   ├── alert.gif     # 警告动画
│   │   └── interact.gif  # 互动动画
│   └── skin2/       # 其他皮肤（可选）
│
├── sounds/          # 音效文件
│   ├── click.wav         # 点击音效
│   ├── alert.wav         # 提醒音效
│   ├── complete.wav      # 完成音效
│   └── hover.wav         # 悬停音效
│
└── icons/           # 图标资源
    ├── tray_icon.png     # 系统托盘图标
    ├── app_icon.ico      # 应用程序图标
    └── task_icon.png     # 任务图标
```

## 图片资源规范

### 宠物动画

- **格式**: GIF 或 PNG 序列帧
- **推荐尺寸**: 128x128 或 256x256 像素
- **背景**: 透明背景
- **帧率**: 10-30 FPS
- **风格**: 可爱、简约

### 所需动画状态

1. **idle.gif** - 待机动画
   - 用途：宠物静止时的默认动画
   - 特点：轻微呼吸、眨眼等细微动作

2. **walk.gif** - 行走动画
   - 用途：宠物在桌面移动时
   - 特点：行走循环动画

3. **run.gif** - 奔跑动画（可选）
   - 用途：快速移动时
   - 特点：比行走更快的动作

4. **sleep.gif** - 睡觉动画
   - 用途：长时间无操作时
   - 特点：闭眼、打鼾等

5. **happy.gif** - 开心动画
   - 用途：任务完成、用户互动时
   - 特点：跳跃、欢呼等

6. **sad.gif** - 伤心动画（可选）
   - 用途：任务过期、提醒时
   - 特点：低头、眼泪等

7. **alert.gif** - 警告/提醒动画
   - 用途：任务到期提醒
   - 特点：跳跃、挥手等引人注意的动作

8. **interact.gif** - 互动动画
   - 用途：用户点击宠物时
   - 特点：眨眼、点头等反馈动作

## 音效资源规范

### 音效文件

- **格式**: WAV 或 MP3
- **采样率**: 44.1 kHz
- **时长**: 1-3 秒
- **音量**: 适中，不刺耳

### 所需音效

1. **click.wav** - 点击音效
2. **alert.wav** - 提醒音效
3. **complete.wav** - 完成任务音效
4. **hover.wav** - 悬停音效（可选）

## 图标资源规范

### 图标文件

- **系统托盘图标**: 16x16、32x32、48x48 像素 (PNG)
- **应用图标**: .ico 格式，包含多尺寸
- **风格**: 简洁、易识别

## 资源获取来源

### 免费资源网站

1. **OpenGameArt** - https://opengameart.org/

   - 支持各种开源协议

2. **itch.io** - https://itch.io/game-assets/free

3. **Kenney.nl** - https://kenney.nl/

4. **Flaticon** - https://www.flaticon.com/

   - 部分免费

## 使用说明

1. 将资源文件放置到对应目录
2. 确保文件命名与代码中的引用一致
3. 注意文件格式和大小限制
4. 测试所有资源是否正常加载

## 注意事项

- 请确保使用的资源符合相应的版权和使用许可
- 商业使用前请仔细阅读资源的使用协议
- 建议使用透明背景的PNG或GIF格式
- 文件大小尽量控制在合理范围内（单个动画<1MB）

## 自定义皮肤

如需添加自定义宠物皮肤：

1. 在 `images/` 目录下创建新文件夹（如 `skin_cat/`）
2. 按照上述规范准备所有动画文件
3. 在配置文件或设置中添加新皮肤选项
4. 测试所有动画是否正常工作
