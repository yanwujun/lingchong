## 1. 宠物动画资源

### 📁 存放位置
```
assets/images/default/
```

### 🎨 必需的动画文件（共5个）

#### 1.1 idle.gif / idle.png - 待机状态
- **用途**：宠物静止时的默认动画
- **建议动作**：
  - 轻微呼吸动画
  - 眨眼
  - 尾巴摆动
  - 轻微身体晃动
- **推荐尺寸**：128x128 或 256x256 像素
- **格式**：GIF（动画）或 PNG（静态）
- **背景**：透明背景 (Alpha通道)
- **帧率**：10-24 FPS
- **循环**：无限循环
- **参考风格**：
  - 可爱卡通风格
  - 像素艺术风格
  - 手绘风格
  - 3D渲染风格

#### 1.2 walk.gif / walk.png - 行走状态
- **用途**：宠物在桌面移动时
- **建议动作**：
  - 行走循环动画
  - 四肢协调运动
  - 轻快的步伐
- **其他要求**：同 idle
- **建议帧数**：4-8帧

#### 1.3 happy.gif / happy.png - 开心状态
- **用途**：任务完成、用户互动、点击时
- **建议动作**：
  - 跳跃
  - 旋转
  - 欢呼姿势
  - 星星闪烁效果
  - 笑脸表情
- **其他要求**：同 idle
- **建议帧数**：6-12帧

#### 1.4 alert.gif / alert.png - 提醒/警告状态
- **用途**：任务到期提醒时
- **建议动作**：
  - 跳跃引起注意
  - 挥手
  - 举牌子/感叹号
  - 闪烁效果
- **其他要求**：同 idle
- **建议帧数**：4-8帧
- **特点**：需要引人注意，但不能太刺眼

#### 1.5 sleep.gif / sleep.png - 睡觉状态
- **用途**：长时间无操作时
- **建议动作**：
  - 闭眼姿势
  - 睡眠呼吸（缓慢起伏）
  - Z字符飘出
  - 蜷缩姿势
- **其他要求**：同 idle
- **建议帧数**：4-6帧

### 🎨 可选的额外动画

#### 1.6 sad.gif - 伤心状态（可选）
- **用途**：任务过期、错误提示时
- **建议动作**：低头、眼泪、失落表情

#### 1.7 interact.gif - 互动状态（可选）
- **用途**：鼠标悬停、点击反馈
- **建议动作**：眨眼、点头、摇头

#### 1.8 eat.gif - 吃东西（可选）
- **用途**：成长系统、喂食功能
- **建议动作**：进食动画

### 📐 技术规格

| 规格项 | 推荐值 | 说明 |
|--------|--------|------|
| 尺寸 | 128x128 或 256x256 px | 正方形 |
| 格式 | GIF 或 PNG | GIF支持动画 |
| 背景 | 透明 (Alpha通道) | 必需 |
| 颜色模式 | RGBA | 支持透明度 |
| 文件大小 | < 500KB/个 | 越小越好 |
| 帧率 | 10-24 FPS | 流畅但不占资源 |
| 色彩 | 柔和、可爱 | 避免过于鲜艳 |

## 6. 音效资源（可选）

### 📁 存放位置
```
assets/sounds/
```

### 🎵 推荐的音效文件

#### 6.1 click.wav - 点击音效
- **用途**：按钮点击、宠物互动
- **时长**：< 0.5秒
- **音量**：适中
- **音调**：清脆、愉快

#### 6.2 alert.wav - 提醒音效
- **用途**：任务提醒
- **时长**：1-2秒
- **音量**：稍大
- **音调**：引人注意但不刺耳

#### 6.3 complete.wav - 完成音效
- **用途**：任务完成
- **时长**：< 1秒
- **音量**：适中
- **音调**：愉悦、成就感

#### 6.4 hover.wav - 悬停音效
- **用途**：鼠标悬停
- **时长**：< 0.3秒
- **音量**：很小
- **音调**：轻柔

### 技术规格
- **格式**：WAV 或 MP3
- **采样率**：44.1kHz
- **比特率**：128-192 kbps
- **声道**：单声道或立体声
- **文件大小**：< 100KB/个

## 11. 常见问题

### Q1: 宠物动画要几帧？
**A**: 建议4-12帧，流畅但不占用太多资源。idle可以4-6帧，复杂动作可以8-12帧。

### Q2: GIF还是PNG？
**A**: 如果要动画效果用GIF，如果是静态图片用PNG。两者都支持透明背景。

### Q3: 尺寸必须是128x128吗？
**A**: 不是必须，但推荐使用正方形。可以是64x64、128x128、256x256等。

### Q4: 可以用视频格式吗？
**A**: 目前不支持，只支持GIF和PNG图片格式。

### Q5: 如何测试资源？
**A**: 将资源放到对应目录，运行 `python main.py` 即可看到效果。
