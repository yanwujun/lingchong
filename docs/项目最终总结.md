## 🚀 如何使用

### 开发环境运行
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行应用
python main.py
```

### 打包发布
```bash
# 使用虚拟环境打包（避免依赖冲突）
pack_with_venv.bat

# 打包完成后
cd dist
DesktopPet.exe
```

---

## ⚠️ 重要提示

### 打包注意事项

1. **中文路径问题**
   - 如果项目在中文路径下打包失败
   - 请复制到英文路径（如 `D:\DesktopPet\`）
   - 打包完成后 dist 文件夹可以在任何路径使用

2. **资源文件**
   - 打包后必须包含 `assets` 文件夹
   - 必须包含 `config.ini`
   - 必须包含 `data` 文件夹

3. **依赖安装**
   - 建议使用虚拟环境打包
   - 避免系统 Python 环境的包冲突
