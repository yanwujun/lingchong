## 🚀 使用方式

### 立即体验

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/desktop-pet.git
cd desktop-pet

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行（无需API Key也可使用大部分功能）
python main.py

# 4. 配置AI（可选）
# 右键宠物 → 💬 AI对话 → ⚙️ → 输入API Key
```
