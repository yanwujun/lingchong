# 桌面灵宠 v0.2.0 - 使用指南

## 🚀 快速开始

### 1. 启动应用
```bash
python main.py
```

### 2. 首次使用
- 宠物会出现在桌面左上角
- 系统托盘显示宠物图标
- 自动创建数据库和备份

## 🎮 基本操作

### 宠物互动
| 操作 | 方法 |
|------|------|
| 移动宠物 | 左键拖动（2秒平滑动画）✨ |
| 宠物互动 | 单击宠物 |
| 打开菜单 | 右键点击宠物或托盘图标 |
| 隐藏宠物 | 右键菜单 → 隐藏 |

## 📝 待办管理

### 添加任务
**方法1**：右键宠物 → 📝 待办事项 → ➕ 添加任务  
**方法2**：打开待办窗口后按 `Ctrl+N` ✨

### 编辑任务 ✨ 新功能
**方法1**：双击任务  
**方法2**：选中任务后按 `Enter`  
**方法3**：选中任务后点击"编辑"按钮

**功能**：
- 自动回填所有字段
- 修改后自动保存
- 列表自动刷新

### 搜索任务 ✨ 新功能
**方法1**：在搜索框输入关键词  
**方法2**：按 `Ctrl+F` 快速聚焦搜索框

**特点**：
- 实时搜索
- 搜索标题和描述
- 显示匹配数量
- 清空搜索框显示全部

### 删除任务
**方法1**：选中任务后点击"删除"按钮  
**方法2**：选中任务后按 `Delete` 键 ✨

### 完成任务
- 选中任务后点击"✅ 完成"按钮

## ⏰ 提醒功能

### 设置提醒
1. 添加/编辑任务时设置截止日期
2. 系统会在截止前15分钟提醒（默认）

### 提醒触发时
- 宠物切换到提醒动画
- 弹出提醒窗口
- 系统托盘通知

### 处理提醒
- **✅ 标记完成** - 完成任务
- **⏱️ 稍后提醒** - 5分钟后再提醒
- **❌ 关闭** - 关闭提醒窗口

## 💾 数据备份 ✨ 新功能

### 自动备份
- 每次启动自动备份数据库
- 备份位置：`backups/` 文件夹
- 文件名：`tasks_backup_20251011_104530.db`
- 自动保留最近7天的备份

### 手动恢复备份
```bash
# Windows
copy backups\tasks_backup_20251011_104530.db data\tasks.db

# 或者直接在资源管理器中复制
```

## 🐛 常见问题

### Q: 应用无法启动？
**A**: 
1. 检查是否安装 PyQt5：`pip install PyQt5`
2. 查看控制台错误信息
3. 确认 Python 版本 >= 3.8

### Q: 图片不显示？
**A**: 运行 `python generate_placeholder_images.py` 生成图片

### Q: 快捷键不工作？
**A**: 确保待办窗口处于激活状态（点击窗口）

### Q: 如何恢复备份？
**A**: 将 `backups` 目录下的备份文件复制到 `data/tasks.db`

### Q: 编辑任务时日期不对？
**A**: 手动选择正确的日期时间

### Q: 搜索没有结果？
**A**: 
1. 检查关键词拼写
2. 尝试搜索部分关键词
3. 清空搜索框显示全部任务
