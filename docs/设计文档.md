## 1. 宠物动画资源

### 📁 存放位置
```
assets/images/default/
```

### 🎨 必需的动画文件（共5个）

#### 1.1 idle.gif / idle.png - 待机状态
- **用途**：宠物静止时的默认动画
- **建议动作**：
  - 轻微呼吸动画
  - 眨眼
  - 尾巴摆动
  - 轻微身体晃动
- **推荐尺寸**：128x128 或 256x256 像素
- **格式**：GIF（动画）或 PNG（静态）
- **背景**：透明背景 (Alpha通道)
- **帧率**：10-24 FPS
- **循环**：无限循环

#### 1.2 walk.gif / walk.png - 行走状态
- **用途**：宠物在桌面移动时
- **建议动作**：
  - 行走循环动画
  - 四肢协调运动
  - 轻快的步伐
- **建议帧数**：4-8帧

#### 1.3 happy.gif / happy.png - 开心状态
- **用途**：任务完成、用户互动、点击时
- **建议动作**：
  - 跳跃
  - 旋转
  - 欢呼姿势
  - 星星闪烁效果
  - 笑脸表情
- **建议帧数**：6-12帧

#### 1.4 alert.gif / alert.png - 提醒/警告状态
- **用途**：任务到期提醒时
- **建议动作**：
  - 跳跃引起注意
  - 挥手
  - 举牌子/感叹号
  - 闪烁效果
- **建议帧数**：4-8帧

#### 1.5 sleep.gif / sleep.png - 睡觉状态
- **用途**：长时间无操作时
- **建议动作**：
  - 闭眼姿势
  - 睡眠呼吸（缓慢起伏）
  - Z字符飘出
  - 蜷缩姿势
- **建议帧数**：4-6帧

### 🎨 可选的额外动画

#### 1.6 sad.gif - 伤心状态（可选）
- **用途**：任务过期、错误提示时
- **建议动作**：低头、眼泪、失落表情

#### 1.7 interact.gif - 互动状态（可选）
- **用途**：鼠标悬停、点击反馈
- **建议动作**：眨眼、点头、摇头

#### 1.8 eat.gif - 吃东西（可选）
- **用途**：成长系统、喂食功能
- **建议动作**：进食动画

### 📐 技术规格

| 规格项 | 推荐值 | 说明 |
|--------|--------|------|
| 尺寸 | 128x128 或 256x256 px | 正方形 |
| 格式 | GIF 或 PNG | GIF支持动画 |
| 背景 | 透明 | Alpha通道 |
| 帧率 | 10-24 FPS | 流畅度 |
| 循环 | 无限循环 | 持续播放 |

## 3. UI界面设计

### 3.1 窗口设计原则

#### 宠物窗口
- **尺寸**: 128x128 像素（可配置）
- **样式**: 无边框、透明背景、置顶
- **交互**: 可拖动、右键菜单

#### 待办窗口
- **尺寸**: 600x800 像素
- **布局**: 
  - 顶部：搜索栏和添加按钮
  - 中部：任务列表（可滚动）
  - 底部：统计信息
- **样式**: 支持亮色/暗色主题

#### 设置窗口
- **尺寸**: 500x600 像素
- **布局**: 标签页分类
- **内容**: 外观、音效、数据、AI设置

### 3.2 组件设计

#### 按钮
- **尺寸**: 最小 80x30 像素
- **圆角**: 4-8px
- **悬停效果**: 颜色加深或阴影

#### 输入框
- **高度**: 30-35 像素
- **边框**: 1-2px
- **圆角**: 4px
- **焦点**: 边框高亮

#### 列表项
- **高度**: 50-60 像素
- **间距**: 2-4px
- **悬停**: 背景色变化

## 5. 字体建议

### 5.1 中文字体

**推荐字体**:
- **Windows**: 微软雅黑 (Microsoft YaHei)
- **macOS**: 苹方 (PingFang SC)
- **Linux**: 文泉驿微米黑 (WenQuanYi Micro Hei)

**备用字体**:
- 思源黑体 (Source Han Sans)
- 黑体 (SimHei)

### 5.2 英文字体

**推荐字体**:
- **系统默认**: Segoe UI (Windows), San Francisco (macOS)
- **等宽字体**: Consolas, Monaco

### 5.3 字体大小

| 用途 | 大小 | 说明 |
|------|------|------|
| 标题 | 18-24px | 窗口标题 |
| 正文 | 12-14px | 主要内容 |
| 辅助 | 10-12px | 提示文字 |
| 按钮 | 13-15px | 按钮文字 |

## 7. 设计规范总结

### 7.1 一致性原则
- 所有窗口使用统一的颜色方案
- 按钮样式保持一致
- 图标风格统一

### 7.2 可用性原则
- 按钮足够大，易于点击
- 文字清晰可读
- 颜色对比度符合WCAG标准

### 7.3 美观性原则
- 简洁现代的设计风格
- 适当的留白
- 流畅的动画过渡
