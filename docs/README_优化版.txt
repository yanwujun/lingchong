    1. 复制项目到英文路径（如 D:\DesktopPet\）
    2. 在新路径运行 pack_with_venv.bat
    3. 打包成功后 dist 文件夹可以在任何地方使用

  Ctrl+N  - 新建任务

  Ctrl+W  - 关闭窗口
  Delete  - 删除任务
  Enter   - 编辑任务

==================================================================
                      ✨ 新增功能
==================================================================

【编辑任务】
  - 双击任务行即可编辑
  - 或选中任务按 Enter
  - 所有字段自动回填

【搜索任务】
  - 在搜索框输入关键词
  - 或按 Ctrl+F 聚焦
  - 实时显示结果

【平滑动画】
  - 宠物移动流畅自然
  - 2秒过渡动画
  - 不再瞬间跳跃

【边界保护】
  - 拖动时自动限制在屏幕内
  - 永远不会找不到宠物

【数据备份】
  - 启动时自动备份到 backups/
  - 保留最近7天
  - 防止数据丢失

【日志系统】
  - 所有操作记录到 logs/
  - 方便问题排查
  - 自动轮转（最大5MB）
