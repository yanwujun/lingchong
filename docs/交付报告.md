## 🎓 技术特点

### 技术栈
- **开发语言**：Python 3.8+
- **GUI框架**：PyQt5 >= 5.15.0
- **数据库**：SQLite3
- **图像处理**：Pillow >= 10.0.0
- **配置管理**：ConfigParser

### 架构设计
- **设计模式**：MVC模式、观察者模式
- **信号机制**：PyQt信号槽
- **模块化**：高内聚低耦合
- **可扩展**：插件式架构预留

### 核心技术
- 透明无边框窗口实现
- 鼠标事件处理（拖拽、点击）
- QMovie动画播放
- QTimer定时任务
- SQLite数据持久化
- ConfigParser配置管理
- QSystemTrayIcon托盘集成

## 📝 使用说明

### 快速开始（3步）

#### 1. 安装依赖
```bash
pip install -r requirements.txt
```

#### 2. 生成图片（首次运行）
```bash
python generate_placeholder_images.py
```

#### 3. 启动应用
```bash
python main.py
```

### 功能限制
1. ⚠️ 编辑任务功能为基础实现
2. ⚠️ 搜索功能框架已搭建，待完善
3. ⚠️ 设置保存后需重启生效
4. ⚠️ 占位图片为表情符号，建议替换

### 平台限制
1. ⚠️ 主要针对Windows平台开发
2. ⚠️ macOS/Linux兼容性未测试
3. ⚠️ 高DPI支持可能需要调整

### 性能限制
1. ⚠️ 单线程设计，大量任务可能影响性能
2. ⚠️ 动画资源较大时可能占用内存
3. ⚠️ 提醒检查间隔固定60秒

**注意**：这些限制不影响MVP核心功能使用，可在后续版本中改进。
