#### 3. 准备资源文件（1-2小时）
- 方案A：从 [DyberPet](https://github.com/ChaozhongLiu/DyberPet) 下载示例资源
- 方案B：从 [OpenGameArt](https://opengameart.org/) 下载免费素材
- 方案C：使用临时占位图测试

#### 4. 实现动画加载（30分钟）
在 `src/pet_window.py` 中完善 `load_animation()` 方法：
```python
def load_animation(self, animation_name):
    animation_path = f"assets/images/default/{animation_name}.gif"
    if os.path.exists(animation_path):
        self.movie = QMovie(animation_path)
        self.pet_label.setMovie(self.movie)
        self.movie.start()
        self.current_animation = animation_name
        return True
    return False
```

#### 5. 整合主程序（1小时）
修改 `main.py`，连接所有模块。
