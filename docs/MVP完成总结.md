## 🎯 核心功能演示

### 功能1：宠物互动
```
操作：左键拖动宠物
效果：宠物跟随鼠标移动

操作：单击宠物
效果：触发互动反馈

操作：右键点击宠物
效果：弹出菜单（待办、设置、隐藏、退出）

自动行为：
- 每10秒自动移动到随机位置
- 每10秒随机切换动画
```

### 功能2：待办管理
```
步骤：
1. 右键宠物 → 📝 待办事项
2. 点击 ➕ 添加任务
3. 填写：
   - 标题："测试任务"
   - 截止时间：选择日期和时间
   - 优先级：高
4. 保存

结果：
- 任务出现在列表中
- 系统托盘显示通知
- 红色标记（高优先级）
```

### 功能3：提醒测试
```
快速测试步骤：
1. 添加任务，截止时间设为 当前时间 + 2分钟
2. 保存任务
3. 等待2分钟
4. 观察：
   - 控制台输出："[提醒系统] 提醒任务: 测试任务"
   - 宠物切换到alert动画
   - 弹出提醒窗口
   - 可以选择：完成、延后、关闭
```
