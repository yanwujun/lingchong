📦 依赖包
================================================================================

必需：
  - PyQt5 >= 5.15.0       # GUI框架
  - Pillow >= 10.0.0      # 图像处理

内置（无需安装）：
  - sqlite3               # 数据库
  - configparser          # 配置管理
  - datetime              # 日期时间
  - os, sys, json         # 系统库

可选：
  - plyer                 # 跨平台通知
  - pygame                # 音频播放
  - pywin32               # Windows API

🚀 使用流程
================================================================================

1. 安装依赖
   pip install -r requirements.txt

2. 生成图片
   python generate_placeholder_images.py

3. 启动应用
   python main.py

4. 测试功能
   - 拖动宠物
   - 右键菜单
   - 添加任务
   - 等待提醒
