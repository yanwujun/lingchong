## 🎯 项目概述

### 产品定位
桌面灵宠是一款集**任务管理**、**时间管理**、**AI陪伴**和**游戏化成长**于一体的智能桌面助手应用。通过可爱的宠物形象和RPG元素，让用户在高效完成工作的同时，获得有趣的养成体验。

### 核心价值
1. **提升效率**: 任务管理+番茄工作法
2. **智能陪伴**: AI对话+图片识别
3. **游戏化激励**: 经验成长+成就系统
4. **趣味养成**: 多宠物+道具系统

## 📝 详细功能需求

### 1. 桌面宠物系统

#### 1.1 宠物显示
**需求描述**: 在桌面上显示可爱的宠物角色

**功能细节**:
- 宠物窗口大小: 128×128像素（可配置）
- 窗口属性: 无边框、置顶、透明背景
- 支持拖动: 可自由移动到桌面任意位置
- 记忆位置: 下次启动恢复上次位置
- 动画支持: idle/walk/sleep/happy/alert 5种状态

**技术实现**:
- 继承 `QWidget`
- 使用 `QLabel` 显示图片
- `setWindowFlags(Qt.FramelessWindowHint | Qt.WindowStaysOnTopHint)`
- 鼠标事件: `mousePressEvent`, `mouseMoveEvent`

**配置项**:
```ini
[Pet]
size = 128
skin = default
always_on_top = True
```

#### 1.2 右键菜单
**需求描述**: 右键宠物显示功能菜单

**菜单项**:
1. 📝 任务管理
2. 🍅 番茄钟
3. 💬 AI对话
4. 🐾 宠物 ▶
   - 🏆 成就
   - 🎒 背包
   - 🛒 商店
5. ⚙️ 设置
6. ℹ️ 关于
7. ❌ 退出

**技术实现**:
- `contextMenuEvent` 事件
- `QMenu` 创建菜单
- 信号连接到对应窗口的show方法

### 2. 任务管理系统

#### 2.1 任务数据结构
**数据库表**: `tasks`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| title | TEXT | 任务标题 |
| description | TEXT | 任务描述 |
| priority | TEXT | 优先级 (高/中/低) |
| status | TEXT | 状态 (待办/进行中/已完成) |
| category | TEXT | 分类 |
| due_date | TEXT | 截止日期 |
| remind_time | TEXT | 提醒时间 |
| created_at | TEXT | 创建时间 |
| completed_at | TEXT | 完成时间 |
| completed_date | TEXT | 完成日期 [v0.4.0] |
| pomodoro_count | INTEGER | 番茄钟数量 [v0.4.0] |

#### 2.2 任务CRUD
**功能**: 创建、读取、更新、删除任务

**创建任务**:
- 输入: 标题（必填）、描述、优先级、分类、截止日期、提醒时间
- 验证: 标题不能为空，日期格式正确
- 标签: 可选多个标签
- 保存: 存入数据库，刷新列表

**编辑任务**:
- 触发: 双击任务行
- 修改: 所有字段可编辑
- 保存: 更新数据库

**删除任务**:
- 触发: 选中任务点击"删除"按钮
- 确认: 二次确认对话框
- 执行: 从数据库删除

**完成任务**:
- 触发: 勾选复选框
- 动作: 更新状态为"已完成"，记录完成时间
- 奖励: 触发宠物成长（+5经验，+10积分）[v0.4.0]

#### 2.3 任务筛选和搜索
**需求**: 快速定位任务

**筛选条件**:
- 状态: 全部/待办/已完成
- 优先级: 全部/高/中/低
- 分类: 全部/工作/学习/生活/其他
- 标签: 按标签筛选 [v0.3.0]

**搜索功能**:
- 实时搜索: 输入关键词即时过滤
- 搜索范围: 标题和描述
- 搜索逻辑: 包含即匹配（不区分大小写）

#### 2.4 任务标签系统 [v0.3.0]
**需求**: 为任务添加多个标签，便于分类和筛选

**数据库表**: `tags`, `task_tags`

**tags表**:
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 标签名 |
| color | TEXT | 标签颜色 |
| created_at | TEXT | 创建时间 |

**task_tags表**:
| 字段 | 类型 | 说明 |
|------|------|------|
| task_id | INTEGER | 任务ID |
| tag_id | INTEGER | 标签ID |

**功能**:
- 创建标签: 输入标签名，自动生成颜色
- 添加标签: 任务对话框中选择/创建标签
- 显示标签: 任务列表中显示标签按钮
- 按标签筛选: 点击标签过滤任务

### 3. 提醒系统

#### 3.1 定时提醒
**需求**: 在指定时间提醒用户完成任务

**触发条件**:
- 任务设置了提醒时间
- 当前时间 >= 提醒时间
- 任务状态为"待办"或"进行中"

**提醒方式**:
- 弹窗提醒: 显示任务标题和描述
- 音效提醒: 播放alert.wav [v0.3.0]
- 宠物动画: 切换到alert状态

**提醒弹窗**:
- 按钮: "完成"、"贪睡5分钟"、"关闭"
- 样式: 跟随主题 [v0.3.0]
- 位置: 屏幕右下角

#### 3.2 贪睡功能
**需求**: 延迟提醒时间

**逻辑**:
- 点击"贪睡5分钟"
- 提醒时间 += 5分钟
- 关闭弹窗
- 5分钟后再次提醒

### 4. 番茄钟系统 [v0.4.0]

#### 4.1 番茄工作法
**需求**: 专注工作25分钟，休息5分钟

**数据库表**: `pomodoro_sessions`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| task_id | INTEGER | 关联任务ID |
| pet_id | INTEGER | 关联宠物ID |
| duration | INTEGER | 时长（秒） |
| session_type | TEXT | 类型（work/break） |
| start_time | TEXT | 开始时间 |
| end_time | TEXT | 结束时间 |
| completed | INTEGER | 是否完成 |

#### 4.2 功能模块

**番茄钟主窗口** (`PomodoroWindow`):
- 工作时长设置: 15/25/30/45分钟
- 休息时长设置: 5/10/15分钟
- 任务选择: 关联当前任务
- 统计显示: 今日完成番茄钟数
- 控制按钮: 开始/暂停/停止

**桌面小组件** (`PomodoroWidget`):
- 悬浮显示: 透明背景，可拖动
- 实时倒计时: MM:SS格式
- 状态指示: 工作中🍅/休息中☕
- 一键控制: 点击暂停/继续

**专注模式**:
- 自动隐藏: 启动番茄钟后自动隐藏其他窗口
- 最小化干扰: 禁用通知
- 完成提醒: 播放音效，显示提示

#### 4.3 奖励机制
**完成番茄钟奖励**:
- +3 经验值
- +5 积分
- 任务 `pomodoro_count` +1

### 5. 宠物成长系统 [v0.4.0]

#### 5.1 经验和等级
**数据库表**: `pets`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 宠物名称 |
| pet_type | TEXT | 类型 |
| level | INTEGER | 等级 |
| experience | INTEGER | 经验值 |
| evolution_stage | INTEGER | 进化阶段 |
| hunger | INTEGER | 饥饿度(0-100) |
| happiness | INTEGER | 心情(0-100) |
| health | INTEGER | 健康(0-100) |
| energy | INTEGER | 能量(0-100) |
| coins | INTEGER | 积分 |
| last_fed_time | TEXT | 最后喂食时间 |
| created_at | TEXT | 创建时间 |
| is_active | INTEGER | 是否激活 |

**升级规则**:
- 每级所需经验: 100点
- 升级时触发: level_up信号
- 升级奖励: 全属性+10

**进化阶段**:
1. 幼年期: 1-9级
2. 成长期: 10-24级（10级进化）
3. 成熟期: 25-49级（25级进化）
4. 完全体: 50级+（50级进化）

#### 5.2 宠物属性
**4维属性**:
1. **饥饿度** (hunger): 0-100
   - 每小时自然-5
   - 喂食+20
   - <20时宠物不开心

2. **心情** (happiness): 0-100
   - 完成任务+5
   - 饥饿时-10
   - <30时效率降低

3. **健康** (health): 0-100
   - 每天自然-2
   - 使用药品+30
   - <20时生病

4. **能量** (energy): 0-100
   - 工作/番茄钟-10
   - 休息+20
   - <10时无法工作

**属性影响**:
- 饥饿>80 且 心情>70: 获得经验+20%
- 健康<30: 获得经验-50%
- 能量<20: 无法开始番茄钟

#### 5.3 经验获取
**获取途径**:
| 行为 | 经验值 | 积分 |
|------|--------|------|
| 完成任务 | +5 | +10 |
| 完成番茄钟 | +3 | +5 |
| 连续签到 | +2/天 | +5/天 |
| 解锁成就 | +10-50 | +20-100 |

### 10. 图片识别系统 [v0.4.0]

#### 10.1 图片识别核心
**技术**: OpenAI Vision API

**数据库表**: `image_tasks`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| pet_id | INTEGER | 宠物ID |
| image_path | TEXT | 图片路径 |
| recognized_text | TEXT | 识别结果 |
| tasks_generated | INTEGER | 生成任务数 |
| created_at | TEXT | 创建时间 |

#### 10.2 识别流程
1. **接收图片**:
   - 拖放图片到宠物窗口
   - 触发 `image_dropped` 信号

2. **处理图片**:
   - 读取图片内容
   - Base64编码
   - 发送到Vision API

3. **解析结果**:
   - 提取文本内容
   - AI分析并提取任务列表
   - 格式化为结构化数据

4. **生成任务**:
   - 为每个识别出的任务创建待办
   - 自动分类和设置优先级
   - 保存到数据库

5. **反馈**:
   - 弹窗显示识别结果
   - 显示生成的任务数量
   - 提供编辑入口

#### 10.3 支持格式
- 图片格式: PNG, JPEG, GIF, WEBP
- 最大尺寸: 20MB
- 内容类型:
  - 手写笔记
  - 印刷文字
  - 表格内容
  - 思维导图
