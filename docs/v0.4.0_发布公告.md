## 💻 快速上手

### 1. 安装应用
```bash
git clone https://github.com/yourusername/desktop-pet.git
cd desktop-pet
pip install -r requirements.txt
python main.py
```

### 2. 配置AI功能（可选）
1. 获取OpenAI API Key
2. 右键宠物 → 💬 AI对话 → ⚙️ 设置
3. 输入API Key并保存

详见：[API配置指南](docs/API配置指南.md)

### 3. 开始使用
- 右键宠物查看所有功能
- 🍅 开始第一个番茄钟
- 💬 与宠物聊天
- 📸 拖放图片试试
- 完成任务看宠物升级！
