# 🎉 欢迎使用桌面灵宠！

## 👋 快速开始（3步）

### 1️⃣ 安装依赖
```bash
pip install -r requirements.txt
```

### 2️⃣ 生成图片（如果还没有）
```bash
python generate_placeholder_images.py
```

### 3️⃣ 启动应用
```bash
python main.py
```

## 🚀 启动后可以做什么？

1. **拖动宠物** - 用鼠标拖到任意位置
2. **右键点击** - 打开菜单
3. **添加待办** - 右键 → 📝 待办事项 → ➕ 添加任务
4. **测试提醒** - 添加一个2分钟后的任务，等待提醒
5. **调整设置** - 右键 → ⚙️ 设置

## ❓ 遇到问题？

### 常见问题
- 应用无法启动？→ 检查是否安装了 PyQt5
- 图片不显示？→ 运行 `python generate_placeholder_images.py`
- 提醒不工作？→ 确认任务设置了截止时间

### 获取帮助
1. 查看 [MVP使用说明.md](MVP使用说明.md)
2. 查看控制台输出的错误信息
3. 参考 [快速开始指南.md](快速开始指南.md) 的常见问题部分
