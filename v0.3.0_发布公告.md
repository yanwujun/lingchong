## 🎯 使用指南

### 启用暗色主题
1. 右键点击桌面宠物
2. 选择"设置"
3. 在"系统设置"标签中找到"主题"
4. 选择"深色"即可

### 查看数据统计
1. 打开待办事项窗口
2. 点击工具栏中的"📊 统计"按钮
3. 查看详细的任务数据分析

### 添加任务标签
1. 创建或编辑任务
2. 在"标签"区域输入新标签名称
3. 点击"+ 添加"按钮
4. 或点击已有标签按钮快速选择

### 调节音效音量
1. 打开设置窗口
2. 找到"提醒设置"标签
3. 拖动"音量"滑块调节
4. 勾选/取消"声音提醒"来启用/禁用音效

### 方式一：源码运行
```bash
git clone https://github.com/yourusername/desktop-pet.git
cd desktop-pet
pip install -r requirements.txt

```

### 方式二：打包EXE（推荐）
```bash

```
